Introducere in HTML5

HTML este un limbaj de markup folosit pentru a descrie documente web (pagini web).

HTML este un acronim provenit de la Hyper Text Markup Language.
Un limbaj de markup foloseste taguri
Documentele HTML sunt formate din mai multe taguri. Fiecare tag descrie alt tip de continut
Prima data, HTML-ul a fost creeat pentru a defini structura unui docuemnt, heading-uri, paragrafe, liste, si asa mai departe, dar cu timpul a ajuns sa fie folosit pentru a formata pagini web.Document HTML StandardMai jos am postat codul pentru un document HTML cat se poate de simplu.
